# Todo App

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

A web application for managing tasks (TODO list), developed using HTML, CSS, and vanilla JavaScript.  
This project was created for learning purposes to practice DOM manipulation, event handling, and frontend project structuring.

## Repository

https://github.com/Federico-Roda97/todo-app

## Demo

Open the `index.html` file in your browser to use the application.

## Features

- Add new tasks
- Display tasks
- Delete tasks
- (Optional) Mark tasks as completed/incomplete
- (Optional) Data persistence using localStorage

## Technologies used

- HTML5
- CSS3
- JavaScript

## Project setup

Clone the repository:

git clone https://github.com/Federico-Roda97/todo-app.git

Navigate into the project folder:

cd todo-app

Open the `index.html` file in your browser.
